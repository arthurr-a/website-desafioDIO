- Tags

No HTML, as tags são elementos de marcação usados para estruturar e formatar o conteúdo de uma página web.

Cada tag é delimitada por colchetes angulares ('< >') e geralmente vem em pares: uma tag de abertura e uma tag de fechamento.

A tag de abertura indica o início de um elemento, enquanto a tag de fechamento, que inclui uma barra (/), indica o fim do elemento.

- Hyperlinks

Hyperlinks no HTML, são elementos que permitem a navegação entre diferentes recursos na web, como páginas, documentos, ou arquivos. Eles são criados usando a tag , que significa "âncora" (anchor). O atributo principal dessa tag é o href, que especifica o destino do link.

Hyperlinks são fundamentais para a navegação na web, permitindo que os usuários se movam facilmente entre diferentes recursos e páginas.

Algumas Curiosidades que encontrei foi:

  1. Tags Sem Fechamento

    2. -Nem todas as tags em HTML precisam de uma tag de fechamento. Algumas são chamadas de "self-closing tags" ou "void elements", um exemplo é o hr>, que precisa inserir uma linha horizontal (/)

  2. Comentários em HTML

    3. Comentários são usados para deixar notas ou desativar partes do código sem que elas apareçam no navegador, o que está dentro dos delimitadores (<.!-- e --.>) não é renderizado pelo navegador.

  3. HTML é Insensível a Maiúsculas e Minúsculas

    4. as tags não diferenciam maiúsculas de minúsculas, embora a prática recomendada seja usar letras minúsculas
